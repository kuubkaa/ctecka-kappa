# Zdrojové fonty

`Roboto-Regular.ttf` a `Roboto-Bold.ttf` jsou originální TrueType soubory stažené
z Google Fonts. Slouží jako vstup pro `npm run font:subset`, který z nich vyrobí
ořezanou verzi do `public/fonts/` (tu appka vkládá do PDF).

Jsou tu commitnuté schválně: díky tomu jde font přegenerovat kdykoli i bez internetu
a výsledek je vždy stejný.
